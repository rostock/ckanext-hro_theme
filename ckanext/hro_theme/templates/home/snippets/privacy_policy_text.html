Using this web application

You can use this web application offered by the municipality of Rostock completely without having to provide any personally-identifying information.

Logging

When you access this web application offered by the municipality of Rostock, no data is logged.

Security

The municipality of Rostock applies technical as well as organisational precautions to this web application in order to protect stored and used information against unintentional or intentional manipulation, loss or deletion and against access of unaffiliated third parties. The precautions are being improved continuously according to technological progress.

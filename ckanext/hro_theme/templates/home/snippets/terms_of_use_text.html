Using the data of the municipality of Rostock

The data owned by the municipality of Rostock and offered via this web application are, unless otherwise specified, made available under the Creative Commons Zero Universal 1.0 Public Domain Dedication (CC0 1.0). This means that you can do what ever you want with the data without having to ask for further permission.

Using the ORKa.MV map

The ORKa.MV map, owned by the municipality of Rostock and offered via this web application, is made available under the Creative Commons Attribution 4.0 International (CC BY 4.0). This means that you are free to copy, distribute and transmit the map, to adapt the map, and to make commercial use of the map as long as you attribute the map in the following manner: © Hanse- und Universitätsstadt Rostock (CC BY 4.0).

Using the other content of the municipality of Rostock

The other content owned by the municipality of Rostock and offered via this web application is made available under CC BY 4.0. This means that you are free to copy, distribute and transmit the other content, to adapt the other content, and to make commercial use of the other content as long as you attribute the other content in the following manner: © Hanse- und Universitätsstadt Rostock (CC BY 4.0).

Using the data of the OpenStreetMap project

The data owned by the contributors of the OpenStreetMap project are made available under the Open Data Commons Open Database License (ODbL). For processing these data the municipality of Rostock uses the open-source software Imposm. In any case, the data has to be attributed in the following manner: © OpenStreetMap (ODbL).

Using the data of the districts and district-like towns of Mecklenburg-Western Pomerania
